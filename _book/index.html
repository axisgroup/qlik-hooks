Qlik Hooks

Qlik Hooks is a JavaScript library that provides custom React Hooks built to interface with the Qlik Sense Associative Engine (QAE). Qlik Hooks are meant to simplify the complex asynchronous nature of the Qlik Engine API into a format easy to use with React. The React Hooks reposity can be found at https://github.com/axisgroup/qlik-hooks, and can be installed in projects via npm:

$ npm install qlik-hooks
